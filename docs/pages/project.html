💬 YouTube Comment Sentiment Analysis

Research Project · NLP | 2025

Project Overview

Analyzed viewer comment data across English-learning YouTube channels to understand what draws ESL audiences and whether the content supports language acquisition.

Technical Highlights

  • Collected and processed comments from 305+ videos via the YouTube Data API, extracting structured fields into a per-channel dataframe.
  • Built an automated batch submission pipeline using the OpenAI Batch API — chunked comments into 400-comment batches, controlled concurrency with MAX_CONCURRENT = 1 to stay within token limits, and tracked job status via a tracking_log.csv.
  • Implemented failure recovery: failed jobs can be re-queued by flipping their log status from fail → pending and re-running the script, with no manual re-work required.
  • Applied bigram frequency analysis after stop-word removal to surface recurring viewer themes per channel.
  • Used multi-label classification (OpenAI API, ~20 label categories) to categorize comments into thematic clusters; aggregated results using like-weighting (log transform) and normalization.
  • Scored comments along 4 learning dimensions using keyword stem matching: Listening, Speaking/Pronunciation, Vocabulary/Grammar, and Confidence/Motivation.

📊 Reddit AI Trend Analysis

Independent Data Pipeline Project | Python & NLP

Project Overview

This project built a scalable text‑processing pipeline to analyze public sentiment and discussion trends around artificial intelligence across Reddit. Using regex‑based cleaning, NLP preprocessing, and topic‑frequency modeling, the system transformed millions of unstructured comments into structured insights about how AI adoption and perception evolve over time.

[ Reddit Comment Processing Flow Diagram ]

Technical Highlights

  • Designed a high‑throughput pipeline for cleaning and normalizing large Reddit comment datasets.
  • Applied NLP techniques to extract sentiment, keyword frequencies, and emerging AI‑related topics.
  • Built trend‑tracking visualizations to reveal shifts in public perception of AI technologies.
  • Implemented regex‑based text normalization to handle noisy, informal, and multilingual content.
