Fusing Language and Chemistry

Multimodal Embeddings for Molecular Property Prediction

Date Jan 2026 - May 2026
Role Data Science Researcher

1. Introduction & The Challenge

In computational chemistry, predicting molecular properties accurately is crucial for drug discovery. Traditional models often rely solely on structural graph inputs, which misses out on the vast context available in scientific literature. This project addresses the gap by asking: Can we fuse textual knowledge with chemical structures to predict molecular properties more accurately?

2. System Architecture

The core concept is a multi-modal fusion layer that integrates structural molecular data with natural language representations. We proposed a pipeline that extracts features from two state-of-the-art models:

  • ChemBERTa: Captures SMILES string geometries and molecular properties.
  • BGE-M3: Generates dense multi-lingual textual embeddings from abstract documentation.

3. Key Results & Impact

By implementing a 5-fold cross-validation regression framework, the fused multi-modal embeddings outperformed structure-only baselines, reducing prediction error by a significant margin.

Model Configuration RMSE (Lower is better) R² Score (Higher is better)
Structure Embeddings Only (ChemBERTa) 0.425 0.74
Multimodal Fusion (ChemBERTa + BGE-M3) 0.312 0.86

4. Conclusion & Key Takeaways

This approach demonstrates the power of utilizing large-scale language embeddings to assist in rigid scientific domain problems. Future work involves scaling the pipeline to Apache Spark clusters for larger datasets.
